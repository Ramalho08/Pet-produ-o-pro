# PET Produção PRO

Protótipo de aplicativo para controle de produção, estoque, vendas e lucro.

## Como usar
Abra o arquivo `index.html` no navegador.

## Valores usados
- Resina: R$ 85/kg
- Barro: R$ 15/kg

## Funções
- Cadastro de produtos
- Cálculo automático de custo e lucro
- Registro de vendas
- Controle básico de estoque
- Dashboard financeiro
- Salvamento no navegador usando localStorage
